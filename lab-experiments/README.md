# Lab & Experiments

A safe space to build, break, and learn. Home for tools, CLI applications,
and anything experimental.

---

## Areas

| Folder | Focus |
|---|---|
| [tools/](./tools/) | Utility scripts and helper tools |
| [cli-apps/](./cli-apps/) | Command-line applications |
| [automations/](./automations/) | Workflow and process automations |

---

## Philosophy

- No polish required — this is a lab
- Experiment freely, fail fast, iterate
- When something graduates to "real", move it to `professional-portfolio/`

---

## Starting a New Experiment

1. Create a subfolder with a descriptive name (e.g., `log-parser/`, `jira-helper/`)
2. Add a minimal `README.md` with the idea and current status
3. Commit early and often — history is useful even in experiments
