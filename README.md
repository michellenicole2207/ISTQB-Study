# Personal Development Workspace

A structured space for continuous learning, hands-on practice, and professional growth.

---

## Sections

| Folder | Purpose |
|---|---|
| [studies-certifications/](./studies-certifications/) | Study notes, labs, and resources for certifications |
| [python-practice/](./python-practice/) | Python exercises, challenges, and small projects |

## Goals

- Work toward certifications (ISTQB CTFL, CKAD, AWS, etc.)
- Build Python fluency through consistent practice
- Develop a portfolio that reflects real skills
- Experiment freely — this is a safe space to break things and learn
